# 🌙 أثير - تطبيق ويب شامل ومستقل

تطبيق ويب عربي متكامل يجمع بين عدة أدوات مفيدة في واجهة واحدة جميلة وسهلة الاستخدام.

## 🎯 المميزات الرئيسية

### 📚 10 أدوات متكاملة:

1. **📄 قارئ PDF** - اعرض وحمّل ملفات PDF مباشرة في التطبيق
2. **📝 مفكرتي** - دوّن أفكارك وملاحظاتك مع حفظ محلي
3. **📖 موسوعة أثير** - معلومات عامة مفيدة وممتعة
4. **💧🏃 متتبع الصحة** - تابع تناول الماء والخطوات اليومية
5. **🎮 لعبة التخمين** - لعبة تفاعلية لتخمين الأرقام
6. **🎨 لوحة الرسم** - رسم وحفظ الرسومات كصور
7. **📅 التقويم بالإيموجيات** - حدد الأحداث بإيموجيات مختلفة
8. **🍅 مؤقت بومودورو** - قسّم وقت العمل بتقنية بومودورو (25/5)
9. **🚨 مركز الطوارئ** - أرقام جهات الاتصال الطارئة المهمة
10. **⚙️ الإعدادات** - خيارات تخصيص التطبيق والنسخ الاحتياطي

## ✨ الخصائص التقنية

✅ **مستقل تماماً** - لا يحتاج إلى إنترنت أو خوادم خارجية
✅ **بدون API** - لا يتطلب مفاتيح أو اتصالات خارجية
✅ **حفظ محلي** - جميع البيانات محفوظة على جهازك (localStorage)
✅ **تصميم عربي** - واجهة RTL (يمين لليسار) كاملة
✅ **رسوميات متحركة** - نجوم وشهب وتأثيرات جميلة
✅ **استجابة الهاتف** - يعمل بسلاسة على جميع الأجهزة والأحجام
✅ **بدون أدوات ذكاء اصطناعي** - تطبيق يدوي بحت

## 💾 تخزين البيانات

جميع البيانات محفوظة محلياً باستخدام `localStorage`:
- **water** - عدد أكواب الماء المشروبة
- **steps** - عدد الخطوات المسجلة
- **appNotes** - الملاحظات اليومية
- **calendarEvents** - أحداث التقويم
- **moonEyeColor** - لون عيون القمر المخصص
- **meteorSpeed** - سرعة الشهب

## 🔧 التخصيص

يمكنك تغيير:
- **ألوان** التطبيق من خلال قيم CSS في `index.html`
- **محتوى الموسوعة** بتعديل البيانات في `script.js`
- **أرقام الطوارئ** حسب دولتك
- **مدة بومودورو** بتغيير القيم (25 و 5 دقائق)
